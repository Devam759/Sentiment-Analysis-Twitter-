## 🚀 Projects Overview

### 1. Twitter Sentiment Analysis (Python)
**Location**: `sentiment-analysis-project/`

A comprehensive machine learning pipeline for Twitter sentiment analysis implementing research-based methodology.

**Features**:
- Complete data preprocessing pipeline
- Multiple ML models (SVM, Random Forest, Logistic Regression, Naive Bayes)
- Advanced NLP analysis (NER, topic modeling, semantic similarity)
- Comprehensive evaluation and visualization
- Interactive demo application

**Tech Stack**: Python, scikit-learn, NLTK, Gensim, pandas, matplotlib

**Quick Start**:
```bash
cd sentiment-analysis-project
python main.py --mode full
```

### 2. Sentiment Analysis Web App (React)
**Location**: `sentiment-analysis-web/`

A modern React + Vite web application that integrates with the Python ML models for real-time sentiment analysis.

**Features**:
- Real-time sentiment prediction interface
- Model comparison dashboard
- Interactive charts and visualizations
- Responsive Material-UI design
- Flask API backend integration

**Tech Stack**: React, Vite, Material-UI, Chart.js, Flask, Python

**Quick Start**:
```bash
cd sentiment-analysis-web
npm run dev                    # Frontend (http://localhost:5173)
cd backend && python app.py    # Backend (http://localhost:5000)
```

## 🛠️ Development Setup

### Prerequisites
- Python 3.8+
- Node.js 16+
- npm or yarn

### Python Project Setup
```bash
cd sentiment-analysis-project
pip install -r requirements.txt
python main.py --mode full
```

### Web Application Setup
```bash
cd sentiment-analysis-web
npm install
npm run dev
```

### Backend Setup (for web app)
```bash
cd sentiment-analysis-web/backend
pip install -r requirements.txt
python app.py
```
